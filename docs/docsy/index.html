主题 | Docsy 常用记录

Docsy 是 Hugo 的一个主题。该主题适合作为产品文档。 Docsy github - https://github.com/google/docsy
Categories:
  • themes

快速熟悉 Docsy 的方法：

  1. 熟悉 Hugo 使用。
  2. 对照官方示例，把官方文档通读一遍。
  3. 善于利用官方文档和 Docsy 源码。

官方文档

  • en - https://www.docsy.dev/docs/ (官方)
  • zh - https://before80.github.io/docsy_docs_with_hugo/

示例 example

  • 官方指引 (使用 Docsy 主题) - https://github.com/google/docsy/tree/main/userguide
  • 官方示例列表 - https://www.docsy.dev/docs/examples/
  • 接口文档示例 - https://github.com/cchesser/docs-as-code/
外观（Look）

设置主题的样式、颜色、站点标识、站点图标等。

封面（Cover）

定制站点的封面页，如：首页、关于页。

布局（Layout）

主题会预先定义好几种页面布局，以应对主题考虑的几种页面场景。

简码（Shortcode）

简码，顾名思义"用于简化操作的代码块"。

搜索（Search）

搜索站点关键字。

部署（Deploy）

将站点发布到网上。
